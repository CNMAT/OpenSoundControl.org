::: {.field-items}
::: {.field-item}
OSC enables interaction between different audio processes, and in
particular between Csound and other synthesis engines. The following
opcodes are available:

\- OSCinit - Start an OSC listener thread.\
- OSClisten - Receive OSC messages.\
- OSCsend - Send an OSC message.
