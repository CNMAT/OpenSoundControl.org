SimpleOSC provides a simple wrap around the OSC python implementation by Daniel Holth (http://wiretap.stetson.edu ). It tries to make OSC as easy to use as possible exposing a very simple API. For example sending a message can be as simple as this
import osc
osc.init()
osc.sendMsg( '/test', 999)
