Music controller for the wiimote. (OSX)
OSC Documentation URL:
http://code.google.com/p/musiccontroller/wiki/OSCWiimote

This piece of software implements sending MIDI messages through a MIDI port and OSC through the network by using a wiimote.

It is based on DarwiinRemote and WiiToMidi but with an extended features for using a larger number of MIDI messages with a small set of buttons and gestures (which is the case of the wiimote).

Acknowledges: DarwiinRemote and WiiToMidi and vvosc (http://github.com/mrRay/vvosc/) developers.
