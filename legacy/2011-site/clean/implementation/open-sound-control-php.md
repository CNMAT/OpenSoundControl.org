::: {.field-items}
::: {.field-item}
A small PHP class which can format and send OSC messages. It has no
parsing capability, so it can only be used for uni-directional
communication (i.e. sending of messages).

Changelog:\
Version 0.2: March 22, 2007. Fixes a bug in timetag encoding for
bundles.\
Version 0.1: 2004 Initial release.
