::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
A .NET (C\#) implementation of OSC using UDP as the transport protocol
(unicast, broadcast, and multicast). It includes support for Osc
Messages and Bundles, and supports the following payload data types:
Int32, Int64, Float, Double, String, and Blob (byte array).

The package includes easy-to-use client and server examples.
