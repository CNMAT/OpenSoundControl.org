::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
ChucK is a new (and developing) audio programming language for real-time
synthesis, composition, performance, and now, analysis - fully supported
on MacOS X, Windows, and Linux. ChucK presents a new time-based,
concurrent programming model that\'s highly precise and expressive (we
call this strongly-timed), as well as dynamic control rates, and the
ability to add and modify code on-the-fly. In addition, ChucK supports
MIDI, OSC, HID device, and multi-channel audio.
