::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
WOscLib is a complete (server and client), easy to use OSC
(OpenSound-Control) library, featuring Object Oriented (C++) design
(modularity), platform- independence, type-safety (especially in
OSC-methods), exception-handling and good documentation (doxygen).
