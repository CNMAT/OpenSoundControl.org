::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
DMX lighting controller with native support for OSC in and out. This
allows easy integration of lighting control with other show control
applications like music, VJ and media server. Since it is a lighting
software, you can use it to convert OSC to Artnet or outputting through
a DMX USB dongle. The TUIO protocol is also supported.
:::
