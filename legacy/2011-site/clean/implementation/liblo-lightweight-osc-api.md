::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
liblo is an implementation of the Open Sound Control protocol for POSIX
systems, started by Steve Harris. liblo is written in C and works on
many platforms including Linux, Mac OS X and Windows. It has support for
all the OSC types, threaded servers, dispatching and timestamped
delivery of bundles. It is released under the GNU General Public
Licence.
