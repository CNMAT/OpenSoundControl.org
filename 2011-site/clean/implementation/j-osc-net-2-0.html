J# OSC is a library for talking the Open Sound Control (OSC) protocol in J#/.NET2.0. It is not, in itself, a usable program. Rather, it is a library designed for building programs that need to communicate over OSC (e.g., SuperCollider, Max/MSP, Reaktor, etc.)

J# OSC is very easy to use: with just a couple of lines you're sending and receiving OSC messages.

It's a port of JavaOSC project
