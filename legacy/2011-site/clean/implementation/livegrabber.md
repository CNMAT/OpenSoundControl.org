::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
The LiveGrabber plugins are a set of Max For Live plugins that send
device parameters, track parameters, clip envelopes and audio analysis
information from Ableton Live to any device on the network that supports
Open Sound Control.
