Starting with version 9.1.2 (latest version: 9.1.6), Logic Pro and Logic Express support OSC in their control surface support. Third parties can create plug-ins which communicate with control surface hardware via OSC rather than MIDI.
Plug-n-Play installation via Bonjour is supported.
Logic already comes with a plug-in for TouchOSC.
For further details, please contact Michael Wong (wong1@apple.com).
