::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
Lily is an Open Source, browser-based, visual programming environment
written in JavaScript. Lily enables users to build programs graphically
by connecting functional modules to fetch and direct the flow of data,
play sound or video, add interactivity or display results. Lily programs
can be shared with other Lily users, installed as Firefox add-ons or run
as standalone apps using XULrunner. Its OSC support is based on JavaOSC.
