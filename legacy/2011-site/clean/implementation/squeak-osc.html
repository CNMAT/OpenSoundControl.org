OSC (Open Sound Control) for Squeak.
One can remote control supercollider and other sound generators implementing OSC using UDP.
Also Squeak can be remote controlled by e.g. Puredata or Eyesweb.
Tests are included

Transport Type:
UDP
Platform:
Any / Platform Independent
Features:
Packet Parsing (Client)
Packet Construction (Server)
Bundle Support
High Speed (> 100 hz packet rate)
Bounded Latency
Bundle Support:
Reads Bundles
Creates Bundles
Type Support:
i: int32
s: string
f: float32
