::: {.field .field-type-text .field-field-description}
::: {.field-items}
::: {.field-item}
Net::OpenSoundControl provides a translation from OSC messages to Perl
datastructures and vice versa. The module is platform independent (see
also the test results for the distribution on CPAN) and includes a
simple server and client.
