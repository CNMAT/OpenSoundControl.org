This project is an implementation of a physical dynamics environment which can be controlled via OSC , so that audio projects such as PureData can create physical objects in a virtual space, and then let them interact with each other, colliding and otherwise moving around. The environment can be used for interacting with a hand controller haptic device .
Data about objects' position and acceleration, for example, can be retrieved and then used to affect some sort of audio synthesis. It was first presented at NIME 2007 .

This is all accomplished via many GPL (or similar) libraries, such as:

  • CHAI 3D , for haptics
  • ODE , for physical dynamics
  • LibLo , for OSC messaging
  • GLUT & FreeGLUT , for graphics (optional)

It is cross-platform, running on Linux, OS X, and Windows. Unfortunately, most haptic devices only provide drivers for Windows, though CHAI 3D does support the SensAble Phantom family of devices under Linux if you have the driver.
